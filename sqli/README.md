# 🔍 SQL Injection (SQLi)

## 📌 Overview
This section documents my hands-on learning and assessment on SQL Injection, a critical web vulnerability that targets database queries through unsanitized input.

## 🧪 Key Concepts Covered
- Authentication bypass using SQLi
- Data extraction techniques
- Understanding backend query manipulation

## 🚨 Impact
- Unauthorized data access
- Database compromise
- Privilege escalation

## 🛡 Mitigation
- Prepared statements (parameterized queries)
- Input validation and sanitization
- Least privilege database access
